# Palette Picker CLI 🎨

A command-line tool that applies retro aesthetics and curated color palettes to your images using the Floyd-Steinberg dithering algorithm.

## Features
* **Maximum Accuracy:** Uses exact vector calculations via NumPy to map pixels to the closest palette colors.
* **Adjustable Smoothness:** Control the diffusion strength to range from smooth, stylized gradients to raw, grainy retro textures.
* **Interactive Mode:** Run without flags for a rich, interactive terminal UI.

## Installation

The recommended way to install Palette Picker globally is using `pipx`.

```bash
# Clone the repository
git clone [https://github.com/gabsdfreitas/palette-picker-cli.git](https://github.com/gabsdfreitas/palette-picker-cli.git)
cd palette-picker-cli

# Install globally
pipx install .
